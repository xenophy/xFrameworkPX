logGamma function

 The natural logarithm of the gamma function. </ br /> Based on public domain NETLIB (Fortran) code by W. J. Cody and L. Stoltz </ br /> Applied Mathematics Division </ br /> Argonne National Laboratory </ br /> Argonne, IL 60439 </ br />   References:  
  1. W. J. Cody and K. E. Hillstrom, 'Chebyshev Approximations for the Natural
     Logarithm of the Gamma Function,' Math. Comp. 21, 1967, pp. 198-203.

  3. Hart, Et. Al., Computer Approximations, Wiley and sons, New York, 1968.
From the original documentation:
This routine calculates the LOG(GAMMA) function for a positive real argument X.  Computation is based on an algorithm outlined in references 1 and 2.  The program uses rational functions that theoretically approximate LOG(GAMMA)  to at least 18 significant decimal digits.
